# 📦 نظام MSP لتوثيق زيارات فريق المبيعات
### MSP Sales Team Visit Documentation System (V3.0)

نظام تقني متكامل وموحد مخصص لتوثيق وتحليل الزيارات الميدانية التي يقوم بها فريق المبيعات في شركة **Modern Style Pack (MSP)**. يهدف النظام إلى رفع كفاءة التوثيق، ضمان دقة البيانات الميدانية، وتحقيق المطابقة المالية الفورية.

---

## 🚀 الأهداف الرئيسية للنظام (Core Objectives)
* **توثيق الزيارات:** تسجيل لحظي لكل زيارة ميدانية مع تحديد الموقع (المحافظة والمنطقة).
* **دقة البيانات:** الربط الديناميكي مع جداول المناطق والقطاعات المعتمدة.
* **الرقابة والشفافية:** نظام سجلات (t98_logs) لرصد نشاط الفريق وتوقيت العمليات.
* **الهوية الموحدة:** واجهة مستخدم احترافية تعكس هوية MSP (الأخضر والبرونزي) مع نظام "Dark Mode".

---

## 🛠️ الهيكلية البرمجية (System Architecture)
يعتمد النظام على بنية **"المحرك الموحد"** لضمان سهولة الصيانة وتوحيد الواجهات:

* **القاعدة الذهبية (Golden Rules):**
    * جميع أسماء الجداول تبدأ بحرف `t` (مثل: `t02_lists`).
    * جميع أسماء الحقول تبدأ بحرف `f` (مثل: `f_visit_status`).
* **التقنيات المستخدمة:** JavaScript (Vanilla), Supabase DB, HTML5, CSS3.

---
